**Technologies Used**

* Javascript
* React
* Node

* MongoDB

### Frontend Technologies:
#### React
Winos frontend was created with Javascript along with the React framework.

### Backend Technologies
#### ExpressJS
Express JS was used to create the server-side of this projet. The backend is not too complex with only a few routes connecting to the frontend.

#### MongoDB
Using MongoDB for the database made creating the tables easy. It is really was the perfect database for this project.
